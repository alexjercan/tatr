# Implement Task Tracker

- STATUS: OPEN
- PRIORITY: 100
- TAGS: feature

Task Tracker is a `TODO` management tool. This is insipired from
<https://github.com/tsoding>'s streams where he uses it for some projects.

This CLI tool should allow us to:
- define new tasks `tatr new`
- query tasks `tatr ls`

We will add extra features to these simple commands as we go. But for now we
need the basics to be able to do stuff.
